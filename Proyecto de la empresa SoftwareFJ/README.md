# Software FJ

## Descripción del proyecto

Este proyecto fue realizado con el propósito de crear un sistema de gestión para la empresa **Software FJ**, donde se pueden administrar clientes, servicios y reservas de una manera organizada.

El sistema fue desarrollado en **Python** utilizando **Programación Orientada a Objetos (POO)**, sin necesidad de usar bases de datos. Para guardar la información se trabajó con objetos, listas y archivos, permitiendo que el programa funcione correctamente y maneje errores sin detenerse.

Durante el desarrollo del proyecto se aplicaron temas vistos en clase como la abstracción, herencia, encapsulación, polimorfismo y el manejo de excepciones, con el fin de crear un sistema más ordenado, funcional y fácil de entender.

## Funciones del sistema

El programa permite:

- Registrar clientes
- Gestionar diferentes servicios
- Crear y procesar reservas
- Detectar errores y continuar funcionando
- Guardar eventos y errores en un archivo de registros

## Servicios disponibles

- Reserva de salas
- Alquiler de equipos
- Asesorías especializadas

## Temas aplicados en el proyecto

- Programación Orientada a Objetos (POO)
- Abstracción
- Herencia
- Polimorfismo
- Encapsulación
- Manejo de excepciones
- Uso de archivos y logs
- Validación de datos

## Cómo ejecutar el proyecto

Para ejecutar el sistema, abrir la terminal y escribir:

python main.py
